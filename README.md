# Jiahe-LearningEdu
# 佳禾教育程序
### 所用技术
##### 前端 
- 前端框架：vue.js
- 前端Ui: elementUI
- 图形组件：百度 echarts
##### 后端
- 核心框架：Spring Boot 2.3
- 数据连接池：hikari
- 定时器：Quartz 2.3
- 安全框架：Apache Shiro 1.4
- 视图框架：Spring MVC 5.0
- 持久层框架：MyBatis 3.5
- 文档工具：swagger2
##### 软件
- JDK1.8
- MySQL5.7
- Maven3.5
##### 微信小程序关联
- UI框架 vant-weapp
